## 2.框架部分

### Vue（🌟🌟🌟🌟）：

推荐先从Vue2学起，结合Vue官方文档食用，看文档才是对培养自学最好的手段！！

了解组件、生命周期、路由以及mixns等的使用等基本的使用，了解Vue的快捷指令，明白双向数据流。

（Vue3和Vue2我的理解是没有真正的优劣，Vue3对于大型项目来说可能更好维护，Composition API的，但是也失去了一定的逻辑性）

了解完Vue2之后，再来看Vue3的新特性，新的生命周期钩子函数

个人推荐视频👇
